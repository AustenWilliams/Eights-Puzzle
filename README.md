# Eights-Puzzle
## Implemented search algorithms to solve MODIFIED 8-puzzle:

### Breadth-first (BFS)
### Depth-first (DFS)
### Uniform-Cost (UCS)
### Greedy Best-first (GBF), with heuristic ℎ1: number of tiles not in correct position
### A* with heuristics ℎ1, ℎ2, and ℎ3
